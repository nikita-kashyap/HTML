HTML

HTML stands for Hypertext Markup Language . Hypertext means readable text and Markup means tostructure it in a specific format.

It is a programming language that is used to design and format of web page. It is the main markup language for creating web pages and other information that canbe display in a web browser.

HTML was developed by TIM BERNERS LEE in the late 1990s . It is a markup language that web browser used to interprate and compose text, image and other material into visual and audiable web page.

Each HTML document is divided into two main parts HEAD and BODY. The Head contains information to identify the web page ,while Body contains actual information to be display.

<table> tag:-

A table tag show the information in tabular form or grid (row&column). A table place information inside the cell form by dividing a rectangle into rows and column. It is used in creating web page layout.

HTML represents a basic table using 4 elements in a table-

(1) <table > tag

(2) <tr > tag:- Represents table row.

(3) <th> tag:- Represents table head.

(4) <td > tag:- Represents table data.


ATTRIBUTES of <table> tag-

(1) align:- This attribute specifies the alignment of the table. Its value is center, left or right.

(2) background:- This attribute specifies the url of a background image for the table. Image is tied if it is smaller than table dimention. <table background="url of the image">...</table>.

(3) bgcolor:- This attribute specifies the background colour for the table . Its value can be either a named colour (such as red,green,blue,aqua etc) or a color specified in hexadecimal (such as #rr0000,#00gg00,#0000bb). <table bgcolor="red">...</table> or <table bgcolor="red+green">...</table> or <table bgcolor="#rr0000">...</table> or <table bgcolor="rgb(255,0,0)">...</table> .

(4) border:- This attribute specifies the width of table border in pixel. A value of 0 makes the borderless table that is useful for graphics layout. The 0 value is very important for defining various diagram or controls in a web page through table tag . <table border="5">...</table> or <table border="0">...</table>.

(5) height and width:- These attributes specify the height and width of the table in pixel or percentage. <table height="500" width="500">...</table> or <table height="100%" width="100%">...</table>.

(6) cellpadding:- This attribute set the width between edge of a cell and its content in pixel. <table cellpadding="10">...</table>.

(7) cellspacing:- This attribute set the width between the individual cells. <table cellspacing="10" >...</table>.

(8) rules:- This attribute controls the display of dividing rules within the table. The value of all specifies dividing rules for rows,cols or groups. <table rules="cols">...</table> or <table rules="rows">...</table> or <table rules="groups">...</table>.

    image tag or <img>

    Image are the most important content of the web site. An image provides more information than would be conveyed with plane text is truely valuable.

    To insert an image into web page, we use image tag i.e., <img> and set the src (source) attribute of element equal to the URL of the image.

    The form of URL can be either an absolute URL or a relative URL .Most likely the image element will use a relatine url to an image found locally.


    To insert a gif image i.e., logo.gif residing in same directory as current document we use -

    Relative URL :-

    <img src="logo.gif" width="200" height="200" />


    we can also use an Absolute URL to reference an image on another server or drive.

    Absolute URL :-

    <img src="http:\\www.bigcompany.com\logo.gif" width="200" height="200"/>

    <img src="D:\images\logo.gif" width="200" height="200" />

    <anchor> tag:-

    Anchor tag defines the hyper links. It is used in HTML to specify a hyper link that link one document to another.

    Anchor element needs two piece of information i.e.,the URL of the target resources and document context required to activate the hyper link.

    The "href" attribute of anchor element specify the target resources for assigning the URL value.

    Most of the hyper links that define use "http url" to link one HTML document to another.

    The simple hyper links combine implicitly , the reference document is the server's home page,which is default document return from web server root directory.

    <a href="url">...(/a)

    Attributes of <anchor> tag:-

    (1) tabindex:- This attribute defines the order in which links will be tabbed through in a browser the support keyboard navigation.

    The value of tabindex is usually a positive number. The browser tab through link user of increasing tabindex value, but skip over those with negative value.

    <a href="marksheet.html" tabindex="3"> title </a>

    (2) accesskey:- This aattribute sets a key to invoke the anchor without requiring a pointing device to select the link.

    The link is activated with the combination of accelerator key i.e., alt and the key specified by this attribute.

    <a href="marksheet.html" accesskey="m" > title </a>

    (3) title:- The title attribute can be used to tooltip information.

    If a user holds the mouse over the link , a tooltip will show the information by the title attribute.

    <a href="marksheet.html" title="BCA 4th sem"< title </a>

    (4) target:- This attribute is used in conjunction with frames to target link such that result loads in perticular frame.

    The target attribute is added to the another element.

    <a href="marksheet.html" target="frame_marksheet"> title </a>

    (5) name:- The name attribute replace 'href' attribute to set a marker the value of name.

    attribute is arbitrary symbolic name of marker that must be unique with the document.

    <a href="#introduction"> www info </a>

      <marquee> Tag
      Marquee tag is used for scrolling piece of text , link or image displayed either horizontally across or vertically across depend upon the setting of web page.

                < marquee> THIS IS MARQUE TAG </marquee>

      Attributes of marquee tag:-

      1. direction :-
        2. This attribute defines the direction of scrolling the content(like image ,text or link). The value is up,down,left or right.

        <marquee direction="up"> THIS IS MARQUEE TAG </marquee>

        In this attribute the text("this is marquee tag") will be scroll to downward to upward direction.
      2. behavior:-
        3. This attribute defines the type of scrolling. The value is scroll or alternate .The default value isscroll.

        <marquee direction="up" behavior="alternate"> THIS IS MARQUEE TAG </marquee>

        In this attribute the text("this is marquee tag") will be either scroll or alternate scroll.
      3. bgcolor:-
        4. This attribute specifies the background colour.

        <marquee direction="up" bgcolor="aqua"> THIS IS MARQUEE TAG </marquee>

      4. scrolldelay:-
        5. This attribute defines the how long to delay between each jump(pass by value in millisecond). This make scrolling slow or fast depend on it's value. Lower the value higher the speed.

        <marquee direction="up" scrolldelay="150"> THIS IS MARQUEE TAG </marquee>

        Event Attributes:-

      5. onmouseover:-
        6. Script runs when mouse-pointer moves over an element(marquee tag).
      6. onmouseout:-
        7. Script runs when mouse-pointer moves out an element(maequee tag).

        <marquee onmouseover="stop();" onmouseout="start();"> THIS IS MARQUEE TAG </marquee>

        Or

        <marquee onmouseover="this.stop();" onmouseout="start();"> THIS IS MARQUEE TAG </marquee>
